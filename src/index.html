ng new $name - to create a project
ng generate - ng g c $name | ng g p $name | ng g s $sname -- to create a component, pipe and service.
ng build - at the end to convert everything to minimised set of files
ng serve - to start the server
ng serve --port portNo -o -> provide our own port to start server and open browser.
ng test - used for testing purpose.
this is Single page Application. only index page is displayed
